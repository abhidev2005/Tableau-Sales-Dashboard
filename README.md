Sales Performance Dashboard

*_📊 Project Overview_*

This project showcases an interactive Sales Performance Dashboard built using Tableau. The objective was to analyze sales data to gain insights into category revenue, monthly trends, regional profitability, and individual product performance.

_*🛠️ Tools & Workflow*_
• Data Extraction: Dataset sourced from Kaggle.
• Data Cleaning: Performed data cleaning and preprocessing using
Google Colab.
• Data Analysis: Executed SQL queries to filter, aggregate, and analyze data for meaningful insights.
• Visualization: Created a comprehensive dashboard using Tableau to present the final findings.

*💡What I Learned Data Visualization:* Learned how to create and customize different types of charts (Bar, Line, and Pie charts) to represent complex data clearly.
• Dashboarding: Mastered the art of arranging visual elements in a grid layout using Tiled containers to create a professional user interface.
• End-to-End Analysis: Gained hands-on experience in the complete data pipeline—from raw data extraction to final visualization.

*📈 Key Insights Category Revenue:* Visualized the revenue contribution of different categories (Accessories, Electronics, and Office supplies).
Monthly Sales Trend: Identified seasonal patterns and fluctuations in monthly sales performance.
Regional Profit: Analyzed profitability across various regions using a Pie chart.
Product Performance: Ranked products based on their total revenue contribution.
